# Audio

Drop a generated ambient track here as **`ambient.mp3`** and the site will use it as the
site-wide background bed automatically (it takes precedence over the built-in synth fallback
in `src/audio/soundscape.js`). Make it a seamless loop; ~60–120s is plenty.

The cap "screw-on" sound is read from the hero video `public/hero-seal.mp4` (its own audio
track) — no file needed here for that.
